# あるある選挙 オンライン版

リアルタイム多人数の「あるある」投稿・投票ゲーム。サーバ（`server.js`）が
唯一の正で、フェーズ・タイマー・投稿・票・採点をすべて管理する。

## 不変条件（変更時は要レビュー）

- **採点エンジン**: `lib/scoring.js` がルールの正実装。仕様 §4 準拠。ロジックを変えない。
- **投票制約**: 1投稿1票・自分の投稿には投票不可・持ち票ちょうどで確定。サーバ側で検証する。
- **匿名性**: 投稿は gallery/voting で匿名・全員共通シャッフル順。投函イベントは投稿者・本文を秘匿。
- **同期演出**: 投函/投票のたびに全員へブロードキャストし、各クライアントで同じ演出を再生する。

## トンマナ（デザイン言語）

配色5色（`--stage / --pink / --cyan / --yellow / --ink` ＋派生）は固定。
形と動きで Discord アクティビティ系のポップなゲームUIを表現する。
キーワード: **ステッカー感・弾む・マスコット**。

**ステッカー影・バックアウトイージングを標準とする。**

### 角丸・ボーダー
- カード/パネル: `border-radius: 18px`
- ボタン・投票カード: `border-radius: 14px`
- 主要要素に太枠: 白カード上は `border: 2.5px solid #241338`、
  夜背景上は `border: 2.5px solid rgba(255,255,255,.9)`

### ステッカー影（ベタ落ち・ぼかし無し）
- カード: `box-shadow: 4px 4px 0 rgba(0,0,0,.35)`
- ボタン: `box-shadow: 0 4px 0 <その色の25%暗色>`
  （pink→`--pink-dark` / cyan→`--cyan-dark` 等）
- ボタン `:active` は沈む押し心地: `transform: translateY(3px)` ＋ `box-shadow: 0 1px 0`

### イージング（出現系アニメ）
- バックアウト（弾む）: `cubic-bezier(.34,1.56,.64,1)`
- カード/リスト項目の出現: fade ＋ `translateY(10px)→0`、各 **60ms** ずらし。
  共通クラス `.pop-in` を各画面のリストに付与する。

### モーション無効化
- `@media (prefers-reduced-motion: reduce)` では出現アニメ・まばたき・揺れを
  すべて無効化する（音SEはこの設定と独立、ミュートボタンのみで制御）。

### その他
- 背景: `radial-gradient(circle at 50% -10%, #3a2160 0%, #241338 55%)` ＋
  紙吹雪ドット柄（インラインSVG data-URI、opacity .06 以下）。外部画像は使わない。
- マスコット「投函箱くん」: `.bbox` に目＋笑い口（インラインSVG）。
  待機中は3s周期まばたき、collect中は左右揺れ（±3deg/1.2s）、投函時は `.pulse`。
- バッジ/ラベル（eyebrow・ROOM・残り票数）は `rotate(-2deg)` のステッカー風。
